# Simple bytecode calculator

A simple bytecode calculator. Can be used with an input file as

```shell
cargo run --realese path/to/mathematical_expression
```

or as a repl:

```shell
cargo run --release

>> 1 + cos(2.3) * sqrt(2e-1)
```
